# Dot Product Netlify Frontend

Standalone frontend prepared for Netlify deployment.

## Local Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```

## Notes

- This frontend is intentionally separate from the Python/Streamlit backend stack.
- It currently uses static demo content.
- To make it interactive, connect it to real backend APIs instead of the existing in-process Streamlit logic.
